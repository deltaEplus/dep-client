# ΔE+ App

React-Native Application for ΔE+

### Requirements

- Node
- Yarn
- Expo CLI

### Setup

* Install Dependencies
    ```
    yarn
    ```    
* To run the App
    ```
    yarn start
    ```
* To build as PWA
    ```
    expo build:web
    ```    
* To run as PWA
    ```
    npx serve web-build
    ```
